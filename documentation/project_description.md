# 📄 SavariGo Flutter — Final Year Project Description

## Project Title
**SavariGo: A Real-Time AI-Based Shared Auto-Rickshaw Pooling System for Urban and Semi-Urban India**

## Technology Stack
| Layer | Technology |
|---|---|
| Mobile App | Flutter (Dart) |
| Database | Supabase (PostgreSQL) |
| Authentication | Supabase Auth + Demo fallback |
| AI Matching | Rule-based Dart algorithm |
| Admin Panel | Flutter screens (in-app) |
| Maps | Custom placeholder UI (Google Maps-ready) |

## Abstract
SavariGo is a Flutter-based mobile application for shared auto-rickshaw pooling in Chennai, Tamil Nadu. It uses a rule-based AI algorithm to match passengers with similar routes, enabling them to share rides and split fares — reducing travel cost by 40–60%, cutting waiting time, and reducing traffic and pollution.

## Problem Statement
- Auto-rickshaws are Chennai's primary last-mile transport
- No affordable AI-based pooling app exists for autos
- Women face safety concerns travelling alone
- Peak-hour congestion is severe in Chennai

## Modules
1. **Passenger Module** — 13 Flutter screens
2. **Driver Module** — 5 Flutter screens  
3. **Admin Module** — 7 Flutter admin screens
4. **AI Matching** — Dart rule-based 100-point algorithm
5. **Safety Module** — SOS, OTP, Guardian Mode
6. **Fare Module** — Auto fare split calculator
